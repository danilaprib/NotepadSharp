## Features
- Save As... dialog
- Delete file or move it to recycle bin
- Basic built-in terminal commands ('ls' and toy 'AWK')
- Calculator that uses expression parser
- Basic text formatting (bold / italic)
- Create / Delete files
- Basic validation for numeric inputs using regex

## Planned features
- Better AWK-like command parsing (and more commands)
- Plugins for syntax highlighting for different programming languages code files
- Spellchecker
- MVVM
- Unit tests
- Installer
- Unit converter
- Search and search&replace functions for text
- Line numbers and Go To Line function
- Themes with possible functionality for users to create custom themes
- Support for multiple formats (docx, pdf, etc.)
